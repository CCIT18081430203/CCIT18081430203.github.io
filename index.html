#8.Hexo常用命令

常用的命令。

  • hexo clean //清除缓存
  • hexo g //生成新数据
  • hexo d //部署
  • hexo s //启动本地服务预览
  • npm install hexo -g //安装Hexo
  • npm update hexo -g //升级
  • hexo init #初始化博客
  • hexo server -p 5000 #更改端口
